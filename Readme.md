# MD-102 Endpoint Administrator Lab

Building and securing the endpoint environment for a simulated company, **Kushaltec**, hands-on in Microsoft Intune and Entra ID. Not slides, actual configuration.

Currently preparing for the MD-102 exam while documenting every phase, mistake, and lesson along the way.

> Note: Kushaltec is a fictional company built in a lab of virtual machines and test users. This documents a simulated small-business endpoint environment, designed and built end to end, not a production deployment.

## The scenario

Kushaltec is a cloud-first Adelaide business, with no on-premises servers. The task: stand up and secure its entire endpoint environment from scratch.
